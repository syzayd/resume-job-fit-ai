## Key Decisions

- **`gemini-2.5-flash-lite` as default model** â€” `gemini-2.0-flash` has 0 quota on the free tier; `gemini-2.5-flash` returns 503s under load. `gemini-2.5-flash-lite` is the reliable free-tier choice. Overridable via `GEMINI_MODEL` env var.
- **Pydantic structured output via `response_schema`** â€” passes the schema directly to Gemini's `GenerateContentConfig`; model returns validated JSON. Fallback `_parse_from_text()` handles edge cases where `.parsed` is None.
- **Session state keying for PDF** â€” tracking `pdf_name` in `st.session_state` prevents re-extracting the PDF on every Streamlit rerun (Streamlit reruns on every widget interaction).
- **`-> NoReturn` for `_handle_api_error`** â€” functions that always raise must be annotated `-> NoReturn` and called with `raise func()` for type checkers to correctly infer that control never returns.
- **XSS via `html.escape()` (stdlib)** â€” used stdlib `html.escape()` instead of a third-party sanitizer; keeps dependencies minimal and is sufficient for escaping Gemini-generated strings before `unsafe_allow_html=True`.
- **Exponential backoff** â€” `time.sleep(4 * (attempt + 1))` gives 4s, 8s before the final attempt. Handles both 429 (rate limit) and 5xx (server) transient errors.
